# AI Resume Builder

A modern and responsive AI Resume Builder built with HTML, CSS, and JavaScript.

## Features

- Live Resume Preview
- AI Summary Generator
- Profile Photo Upload
- Multiple Resume Templates
- Experience & Education Sections
- Skills & Languages
- Projects & Certifications
- GitHub & LinkedIn Links
- PDF Download
- Dark Mode
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
